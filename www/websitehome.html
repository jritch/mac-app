About

Age-CAP is a smartphone and web application which aims to create a crowd-sourced database of age-friendly locations. It consists of interactive forms which enable users to quickly rate a location. Users are also able to browse this database to assess the age-friendliness of locations in their neighborhood, providing them access to information which is otherwise unavailable.

Contact
